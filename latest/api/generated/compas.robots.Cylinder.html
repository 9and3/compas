Cylinder

class compas.robots.Cylinder(radius, length)[source]

Bases: compas.robots.model.geometry.BaseShape

3D shape primitive representing a cylinder.

Parameters

  • radius (str or float) – The cylinder’s radius.

  • length (str or float) – The cylinder’s length.

Attributes

  • radius (float) – The cylinder’s radius.

  • length (float) – The cylinder’s length.

  • geometry (compas.geometry.Cylinder) – The COMPAS geometry of this cylinder.

Examples

>>> c = Cylinder(1, 4)
>>> c.geometry
Cylinder(Circle(Plane(Point(0.000, 0.000, 0.000), Vector(0.000, 0.000, 1.000)), 1.0), 4.0)
